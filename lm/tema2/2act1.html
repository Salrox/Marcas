Actividad 1 Diferencias entre los atributos title y alt

Respuesta

El elemento title se utiliza para crear un toolip, es decir, una ventana pequeña que aparece cuando se posiciona el ratón sobre un texto.

Mientras el elemento alt muestra un texto alternativo que se utilizará cuando no se pueda cargar correctamente una imagen.
